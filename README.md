> [!IMPORTANT]
> Проект предназначен для Windows и требует прав администратора, потому что работает с сетевым трафиком через WinDivert. Используйте его только там, где это не нарушает законодательство, правила сети и условия вашего провайдера.

## ✨ Возможности

- 🎮 готовые фильтры для Discord, включая голосовой трафик и STUN;
- 📺 стратегии для YouTube, Google и других доменов из пользовательских списков;
- 🧩 20 готовых вариантов запуска для разных сетей и провайдеров;
- ⚙️ установка выбранной стратегии как службы Windows;
- 🧪 автоматическое сравнение конфигураций в обычном и DPI-режиме;
- 🩺 встроенная диагностика конфликтующих служб, прокси, DNS и WinDivert;
- 🗂️ отдельные пользовательские списки доменов и IP-адресов;
- 🔄 обновление IPSet, hosts-списка и проверка новой версии.

## ⚡ Быстрый старт

1. Скачайте проект и **полностью распакуйте архив** в отдельную папку.
2. Нажмите правой кнопкой на `general.bat` и выберите **«Запуск от имени администратора»**.
3. Проверьте Discord, YouTube и остальные нужные ресурсы.
4. Если результат нестабилен, остановите текущий `winws.exe` и попробуйте один из файлов `general (ALT...).bat`.
5. Чтобы подобрать конфигурацию автоматически, откройте `service.bat` и выберите **Run Tests**.

> [!TIP]
> Универсальной стратегии для всех провайдеров нет. Начните с `general.bat`, затем пробуйте `ALT`, `SIMPLE FAKE` и `FAKE TLS AUTO`. Лучший вариант — тот, который стабильно работает именно в вашей сети.

## 🧠 Какие стратегии есть

| Группа | Для чего подходит |
|---|---|
| `general.bat` | Основная сбалансированная конфигурация |
| `general (ALT...).bat` | Альтернативные комбинации desync, split и fake-пакетов |
| `general (SIMPLE FAKE...).bat` | Более простые fake-стратегии |
| `general (FAKE TLS AUTO...).bat` | Варианты с автоматической подменой TLS ClientHello |

Названия отражают семейство стратегии, а не рейтинг. Более высокий номер `ALT` не означает, что конфигурация лучше предыдущей.

## 🛠️ Работа как служба

Запустите `service.bat`. Скрипт сам запросит повышение прав и откроет менеджер:

| Раздел | Действие |
|---|---|
| **Install Service** | Выбрать BAT-конфигурацию и установить её как службу `zapret` |
| **Remove Services** | Остановить и удалить `zapret` и связанные службы WinDivert |
| **Check Status** | Проверить службу, драйвер и процесс `winws.exe` |
| **Game Filter** | Включить или отключить игровые TCP/UDP-порты |
| **IPSet Filter** | Переключить режим фильтрации по IPSet |
| **Auto-Update Check** | Управлять фоновой проверкой версии |
| **Run Diagnostics** | Найти типовые конфликты и проблемы системы |
| **Run Tests** | Сравнить доступные стратегии по тестовым адресам |

После смены фильтра или стратегии перезапустите `zapret`, чтобы настройки применились.

## 🧪 Тестирование конфигураций

`utils/test zapret.ps1` умеет запускать стратегии по очереди, проверять HTTP/TLS-доступность и сохранять результаты в папку `test results`.

Список целей настраивается в `utils/targets.txt`:

```ini
DiscordMain = "https://discord.com"
YouTubeWeb  = "https://www.youtube.com"
GoogleDNS   = "PING:8.8.8.8"
```

Доступны два сценария:

- **Standard** — HTTP/1.1, TLS 1.2, TLS 1.3 и ping;
- **DPI** — расширенная проверка реакции сети на разные варианты запроса.

> [!WARNING]
> Во время теста процесс `winws.exe` перезапускается много раз. Не запускайте параллельно другую копию zapret или похожий DPI-bypass.

## 🗂️ Пользовательские списки

При первом запуске автоматически создаются файлы:

- `lists/list-general-user.txt` — дополнительные домены;
- `lists/list-exclude-user.txt` — домены, которые не нужно обрабатывать;
- `lists/ipset-exclude-user.txt` — исключённые IP-адреса и подсети.

Добавляйте по одной записи на строку. После изменения списков перезапустите программу или службу.

## 🩺 Диагностика

Если обход не работает, откройте `service.bat` → **Run Diagnostics**. Проверяются:

- состояние Base Filtering Engine и WinDivert;
- конфликтующие DPI-bypass-службы;
- системный прокси, VPN и Secure DNS;
- AdGuard, Killer, SmartByte, Check Point и Intel Connectivity Network Service;
- записи YouTube в системном файле `hosts`;
- кэш Discord и состояние TCP timestamps.

Не отключайте Microsoft Defender или другой антивирус только ради запуска. Используйте файлы из доверенного источника и проверяйте контрольные суммы бинарников после обновления.

## 📁 Структура проекта

```text
zapret/
├── bin/                     # winws, WinDivert и fake-пакеты
├── lists/                   # домены, IPSet и исключения
├── utils/
│   ├── targets.txt          # цели автоматических тестов
│   └── test zapret.ps1      # тестовый раннер
├── general*.bat             # готовые стратегии запуска
├── service.bat              # менеджер службы и диагностики
└── README.md
```

## 🧹 Как удалить

1. Запустите `service.bat` от имени администратора.
2. Выберите **Remove Services**.
3. Убедитесь, что `winws.exe` больше не запущен.
4. После этого папку проекта можно удалить.
